PROGRAM TITLE
   Where is my Cheese ?
DESCRIPTION:
   'Where is my Cheese ?'라는 이름의 미로탈출 게임을 오픈프레임웍스 기반으로 개발하였다. 


INSTALLING the source: ★★★★★★
   1) 첨부된 압축파일 '20171643 박준혁 기말프로젝트'을 압축 해제한다.
   2) 해제된 압축파일 내의 src 폴더를 들어간다.
   3) src 폴더 내의 5개의 cpp 및 헤더파일(main.cpp, ofApp.cpp, ofApp.h, ofxWinMenu.cpp, ofxWinMenu.h)을 임의의 오픈프레임웍스 솔루션 폴더의 src 폴더 내에 그대로 복사한다.
   4) 비주얼 스튜디오를 통해 소스코드를 복사해놓은 솔루션 파일을 연 후, 해당 소스 파일들을 프로젝트에 포함시킨 후 컴파일하여 정상적으로 구동되는지 확인한다.


REQUIREMENTS: ★★★★★★
   'Where is my Cheese ?' 프로그램은 적합한 GUI를 구현하기 위해 각종 이미지파일과 사운드파일을 참조한다. 따라서 프로그램의 정상 실행을 위해서는 다음의 작업이 가장 중요하다. 

   1) 첨부된 압축파일 '20171643 박준혁 기말프로젝트' 내의 bin 폴더를 들어간다.
   2) bin 폴더 내의 'rank.txt', 'maze.maz' 파일과 data 폴더를 복사한다.
   3) 복사한 두 파일을 그대로 정확하게 '실행하고자 하는 솔루션 폴더의 bin 폴더' 안에 복사한다. (즉, data 폴더의 경우 오버라이드 되는 것이다.)
   

   ※ bin 폴더 내의 'rank.txt' 파일과 'maze.maz' 파일의 경우 실수로 복사되지 않더라도 프로그램의 최초 실행에 문제를 일으키지는 않는다. ★★★
   ※ 하지만 bin 폴더 내의 data 폴더는 정확하게 그대로 복사되어야한다. data 폴더 내의 파일이 하나라도 빠지게 되면 프로그램이 정상적으로 동작하지 않게 된다. (정말 중요한 부분) ★★★
